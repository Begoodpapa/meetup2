DocumentationIL Libgen Process system, test the process related function points provided by libgen. A simulator named "lbgprtst" is implemented. The test cases invoke dmxsend tool to send different c_test_msg(message id 6034) to simulator which lead to different interface calling of libgen.The simulator's master process has two hand group with each group own 20 and 10 hands. This shall be corresponding to the setting in config.ini file.\n
